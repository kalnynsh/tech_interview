3 В каком регистре лучше писать HTML-код? HTML-код лучше писать в нижнем регистре
4 Как убрать синюю рамку вокруг картинки-ссылки? <a href="#" class="link-with-img">
<img scr="#" />
</a>
<style>
.link-with-img {
text-decoration: none;
}
</style>
CSS text-decoration - добавляет оформление текста:
line-through - создает перечёркнутый текст;
overline - линия проходит над текстом;
underline - устанавливает подчёркнутый текст;
none - отменяет все эффекты, в том числе заданное по умолчанию.
5 Обязательно ли писать alt в <img>? alt - альтернативный текст для изображения, указывать обязательно для получения информации при отключении отображений изображений, или для посковых систем, или экранными ридерамию
6 Что такое html-entities html-entities - используемые HTML спецсимволы и дополнительные символы (т.к. §, © и проч.), которые заменяются либо &entity_name; либо &#entry_number; < или <, > или >,   или  , ‑, & &, " или ", ' или '
7 Как убрать подчеркивание для всех ссылок на странице? Присвоить в CSS таблице элементу <a> стиль {text-decoration: none;} в начале таблицы.
8 Что делает свойство clear? CSS свойство clear - отменяет действие float с указанной стороны для вышестоящих элементов.
clear: none | left | right | both
none Отменяет действие clear. Обтекание согласно св-ва float
both Отменяет действие float одновременно с лева и справа.
left Отменяет действие float со значением left.
right Отменяет действие float со значением right.
