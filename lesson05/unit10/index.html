Задание 10

Перечислить по пять знакомых вам псевдоклассов и псевдостилей с указанием их функционала.

Псевдоклассы и псевдостили

№ п/п Наименование Описание
10.1 Псевдо классы ссылки (anchor pseudo-classes)
a:link отображение ссылки, по которой еще не переходил пользователь
a:visited отображение ссылки, по которой уже переходил пользователь
a:hover отображение ссылки или др. HTML элемента, при наведении курсора
a:active отображение выбранной ссылки
- Псевдо классы
Селектор:first-child,
Селектор:last-child
p:first-child соответсвует первому элементу из группы, имеющий общего родителя
p:last-child соответсвует последнему элементу из группы, имеющий общего родителя
- Псевдо классы
Селектор:nth-child(odd|even|<число>|<выражение>)

Псевдокласс :nth-child используется для добавления
стиля к элементам на основе нумерации в дереве элементов.

odd Все нечётные номера элементов.
even Все чётные номера элементов.
<число> Порядковый номер дочернего элемента относительно своего родителя.
Нумерация начинается с 1, это будет первый элемент в списке.
<выражение> Задаётся в виде an±b, где a и b — целые числа, а n — счётчик,
который автоматически принимает значение 0, 1, 2...
- Псевдо классы
Селектор:nth-of-type(odd|even|<число>|<выражение>)

Псевдокласс :nth-of-type используется для добавления
стиля к элементам указанного типа на основе нумерации в дереве элементов.

odd Все нечётные номера элементов.
even Все чётные номера элементов.
<число> Порядковый номер дочернего элемента относительно своего родителя.
Нумерация начинается с 1, это будет первый элемент в списке.
<выражение> Задаётся в виде an±b, где a и b — целые числа, а n — счётчик,
который автоматически принимает значение 0, 1, 2...
10.2 Псевдо элементы (pseudo elements)
p::after вставка контента после p элемента
p::before вставка контента до p элемента
p::first-letter выбор первой буквы у каждого p элемента
p::first-line выбор первой строки у каждого p элемента
p::selection выбор выделенного пользователем текста в p элементе
