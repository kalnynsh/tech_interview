Что такое <ol>, <ul> и <li>?

<ol>

HTML элемент <ol> (ordered list - упорядочный список) отображает на странице нумерованный список.
Каждый элемент списка должен начинаться с <li>.
Если к <ol> применен CSS стиль, то элемент <li> наследует эти стили.

Атрибут type - определяет тип нумерации.

  • type="A" - заглавные латинские буквы;
  • type="a" - строчные латинские буквы;
  • type="I" - заглавные римские цифры;
  • type="i" - строчные римские цифры;
  • type="1" - арабские цифры (по умолчанию);

<ul>

HTML элемент <ul> (unordered list - неупорядочный список) отображает на странице маркированный список.

  • Пункт первый маркированного списка
  • Пункт второй маркированного списка
      • Пункт третий
    • Вложенный пункт маркированного списка
  • Пункт n ..

<li>

HTML элемент <li> (list item - элемент списка) отображает на странице элемент списка.
Родительским элементом устанавливается тип списка: маркированный <ul> или нумерованный <ol>.

<dl> Description list - список описаний

Список описаний состоит из двух элементов — термина и его описания. Сам список задаётся с помощью элемента <dl> (от англ. Description List), термин — элементом <dt> (от англ. Definition Term), а его описание — с помощью элемента <dd> (от англ. Description).
